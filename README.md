# BackendPAM

## Objective

The goal of this project is to build the backend foundation for **PAM – Paulo Azevedo Mobiliário**, a furniture store system.  
It manages products, categories, contacts, and social media information using a PostgreSQL database and a Java-based DAO structure.

---

## Technologies Used

- **Java**
- **PostgreSQL**
- **JDBC (Java Database Connectivity)**
- **Maven**
- **IntelliJ IDEA**
